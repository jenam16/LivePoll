# LivePoll

A real-time polling application built using the MERN stack.

## 🚀 Tech Stack

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose

### Frontend (In Progress)
- React
- Vite

## Features

### ✅ Completed
- Create Poll API
- Get All Polls API
- Get Single Poll API
- Vote API
- Delete Poll API
- MongoDB Atlas Integration

### 🚧 In Progress
- React Frontend
- Poll Creation UI
- Voting UI
- Real-time updates using Socket.IO
- Deployment

```bash
git clone <repo-url>

npm install

npm run dev
```
